# Suriname Mining Risks Visualization Platform

![Project Banner](IMG_20250222_004110.jpg)

A web-based platform visualizing mining risks in Suriname, including legal/illegal mining sites, pollution gradients, and mercury concentration zones.

## Features

- Interactive Leaflet.js map with mining site markers
- Pollution gradient visualization
- Mercury concentration heatmap
- User authentication system (login/register)
- Contact form for user feedback
- News updates and data disclaimer panels

## Prerequisites

- Modern web browser (Chrome, Firefox, Edge)
- Node.js (for development)
- Python (for CSV processing, optional)
